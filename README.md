# Skin Disease Classification

Developed a CNN-based Skin Disease Classification system using Python, TensorFlow, and OpenCV. Preprocessed and augmented image datasets, trained and evaluated the model using accuracy and confusion matrix metrics, and improved classification performance through data optimization and model tuning.

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy
- Pandas
- Matplotlib

## Features
- Image preprocessing
- Data augmentation
- CNN model training
- Disease classification
- Performance evaluation
